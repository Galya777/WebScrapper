# Уеб Скрейпър Приложение

## Описание

Това е Java приложение за уеб скрейпинг, което може да работи както в клиентски, така и в сървърен режим. Приложението е проектирано да извлича данни от уеб страници паралелно, използвайки множество нишки за по-добра производителност.

## Проблем, който решава

Приложението решава проблема с бавното извличане на данни от уеб страници. Чрез използването на паралелна обработка, то значително намалява времето, необходимо за скрейпиране на голям брой елементи от уеб страници.

## Възможни приложения

- Събиране на данни за пазарни проучвания
- Мониторинг на цени в реално време от различни електронни магазини
- Агрегиране на информация от новинарски сайтове
- Анализ на данни от социални мрежи
- Автоматично извличане на контактна информация от уебсайтове

## Използвани технологии

- Java 20+
- Maven за управление на зависимостите
- Jsoup за уеб скрейпинг
- Java NIO за мрежова комуникация
- Gson за сериализация/десериализация на JSON

## Инсталация и стартиране

1. Клонирайте хранилището:
   ```bash
   git clone <repository-url>
   cd mpr2025_KN_FN_java_linux
   ```

2. Създайте JAR файл:
   ```bash
   mvn clean package
   ```

3. Стартирайте сървъра:
   ```bash
   java -jar target/mpr2025_KN_FN_java_linux-1.0-SNAPSHOT.jar server
   ```

4. В друг терминал, стартирайте клиента:
   ```bash
   java -jar target/mpr2025_KN_FN_java_linux-1.0-SNAPSHOT.jar client
   ```

## Архитектура и дизайн

Приложението използва клиент-сървър архитектура с три различни вида сървъри:

1. **Базов сървър (Server.java)** - имплементира неблокиращ I/O с Java NIO
2. **ThreadPool сървър (ThreadPoolServer.java)** - използва пул от нишки за обработка на заявките
3. **Selector сървър (SelectorServer.java)** - използва Java NIO Selector за мултиплексиране на I/O операциите

## Производителност

Приложението включва инструмент за бенчмарк (`Benchmark.java`), който позволява тестване на производителността с различен брой нишки. За да го използвате:

```bash
mvn compile
exec:java -Dexec.mainClass="bg.university.mpr2025.benchmark.Benchmark" -Dexec.args="https://example.com results.csv 100 16"
```
